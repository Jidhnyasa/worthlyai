# Extension Icons

Place three PNG icon files here:

| File        | Size    |
|-------------|---------|
| icon16.png  | 16×16   |
| icon48.png  | 48×48   |
| icon128.png | 128×128 |

Use the Worthly amber logo mark (`hsl(32 95% 54%)` background with the W waveform path).

Until real icons are added, Chrome will display a default grey icon in the toolbar.
To add the icons to the manifest, update the `action.default_icon` and top-level `icons` fields in `manifest.json`.
